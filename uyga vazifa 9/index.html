    2. Soxta akkauntlar va botlar

  2. Profilida rasm/ma’lumot yo‘q bo‘lsa — ehtiyot bo‘lBegona linklarni bosmaSettings → Privacy → Calls / Messages ni faqat kontaktlarga qo‘y

    3. Maxfiylik muammosi

  3. Settings → Privacy → Phone Number → NobodyOxirgi ko‘rilgan vaqtni My Contacts qilIkki bosqichli himoya: Settings → Privacy → Two-Step Verification

    Chrome muammolari

    Chrome sekin ishlaydi

  1. Keraksiz tablarni yop Settings → Performance → Memory Saver ni yoq Settings → Privacy → Clear browsing data (Cache)

    2. Brauzer o‘z-o‘zidan yopilib ketadi

  2. Chromeni yangila chrome://extensions → hamma extension’ni o‘chirib tekshir

    3. Reklama va popup’lar ko‘payib ketgan

  3. Settings → Privacy → Site settings → Pop-ups → Don’t allow Shubhali extension’larni o‘chir Chrome’da Reset settings qil
